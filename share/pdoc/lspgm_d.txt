   fmask    - MAKES 1. OR 0. MASK FROM A GRID                            1 1 C

   ggdev    - GRID SET DEVIATION FROM GLOBAL MEAN                        1 1
   ggdlat   - LATITUDE DERIVATIVE OF A GRID SET                          1 1
   ggdlon   - LONGITUDE DERIVATIVE OF A GRID SET                         1 1
   gghems   - SPLITS GLOBAL GAUSSIAN GRID FILE INTO HEMISPHERES          1 2   G
   gglob    - CONVERTS HEMI GRID OR CROSS-SECTION FILE TO GLOBAL         1 1 C
   ggtrig   - GRID SET MULTIPLIED BY CONST*FUNCTION(LAT,LON)             1 1 C
   ggvecps  - CONVERTS LAT-LONG U,V TO P.S. WIND COMPONENTS              2 4 C

   gpasg    - INTERPOLATES PRESSURE FILES TO SIGMA/ETA/GEM               3 1 C
   gpdpd    - COMPUTES DEW POINT DEPRESSION FROM T AND SHUM OR RHUM      2 1
   gphum    - CONVERTS PRES. GRIDS OF ES,T TO SPECIFIC AND REL HUM       3 2
   gptavt   - COMPUTE VIRTUAL TEMPERATURE FROM TEMPERATURE               2 1
   gpvtat   - CONVERT VIRTUAL TEMPERATURE TO TEMPERATURE                 2 1
   grset    - CREATE SETS OF MULTILEVEL FIELD SET TO AN ASSIGNED VALUE   0 1 C
   gsapl    - INTERPOLATES SIGMA FILE TO PRESSURE                        2 1 C
   gsaptp   - INTERPOLATES SIGMA FILE OF TEMP TO PRESSURE PHI AND TEMP   3 2 C
   gsapzl   - INTERPOLATES SIGMA FILE OF PHI TO PRESSURE PHI AND TEMP    3 2 C
   gsasg    - INTERPOLATES SIGMA/ETA/GEM TO SIGMA/ETA/GEM                3 1 C
   gsathl   - INTERPOLATES A SIGMA/PRESSURE/ETA FILE TO THETA            3 1 C
   gsathlt  - INTERPOLATES FROM ETA OR PRESSURE LEVELS TO THETA LEVELS   4 4 C
   gshumh   - CONVERTS ETA GRIDS OF GCM/RCM ES TO SPECIFIC AND REL HUM   4 2 C
   gsmslp   - COMPUTES MEAN-SEA-LEVEL PRESSURE FROM SIGMA T, LNSP, PHIS  3 1 C
   gsmslph  - COMPUTES MEAN-SEA-LEVEL PRESSURE FROM HYBRID T, LNSP, PHIS 3 1 C
   gsomgah  - CONVERTS ETA (SIGMA/HYBRID) Q,D FILES TO VERTICAL MOTION   4 1 C G
   gsomega  - CONVERTS Q,D FILES TO VERTICAL MOTION                      3 1 C G
   gwtqd    - COMPUTES SPECTRAL Q,D FROM GRID WINDS                      2 2 C
   laslon   - ADD/REMOVE LAST LONGITUDE COLUMN                           1 1 C 
   llagg    - CONVERT LAT-LONG GRID FILE TO GAUSSIAN FILE                1 1 C
   llvagg   - CONVERT LAT-LONG WINDS FILE TO GAUSSIAN FILE               2 2 C
   makeyy   - INTERPOLATE SCALAR FIELDS ONTO A GEM/DM YY-GRID            1 1 C
   makeyyv  - INTERPOLATE VECTOR FIELDS ONTO A GEM/DM YY-GRID            2 2 C
   makezg   - INTERPOLATE SCALAR FIELDS ONTO A GEM/DM Z-GRID             1 1 C
   makezgv  - INTERPOLATE VECTOR FIELDS ONTO A GEM/DM Z-GRID             2 2 C
   meriavg  - COMPUTES MULTI-LEVEL MERIDIONAL AVERAGE OF A SET OF GRIDS  1 1
   mslpr    - COMPUTES MEAN-SEA-LEVEL-PRESSURE FROM T,PHI PRESSURE FILES 2 1
   nhall    - CONVERT NH GAUSSIAN GRID FILE TO LAT-LONG FILE             1 1 C
   psagg    - CONVERT POLAR-STEREOGRAPHIC SCALAR GRIDS                   1 1 C
   psvagg   - CONVERTS POLAR-STEREOGRAPHIC WINDS                         2 2 C
   psgrid   - CALCUL LES PARAMETRES POUR UNE GRILLE POLAIRE STEREO       1 1
   rangrd   - CREATES MULTI-LEVEL SETS OF RANDOM VALUE GRIDS             0 1 C
   rzonavg  - TAKES THE REPRESENTATIVE ZONAL AVERAGE OF A SCALAR FIELD   2 1
   rzonvavg - TAKES THE REPRESENTATIVE ZONAL AVERAGE OF A VECTOR FIELD   3 2
   signag   - PRODUCES A GRID OF (-1,0,1) FROM (-,0,+) INPUT CARDS       0 1 C
   shall    - CONVERT SH GAUSSIAN GRID FILE TO LAT-LONG FILE             1 1 C
   smooth   - SIMPLE MINDED GEOGRAPHICAL SMOOTHER                        1 1
   solflux  - RE-COMPUTE THE INCIDENT SOLAR FLUX AT THE TOP OF THE MODEL 1 1 C
   subarea  - COMPUTES SUB-AREAS (EITHER PS OR LL) OF GG GRIDS           1 1 C
   transpos - TRANSPOSE LATITUDES AND LONGITUDES IN A GRID FILE          1 1
   zonavg   - COMPUTES MULTI-LEVEL ZONAL AVERAGE OF A SET OF GRIDS       1 1
   zoncov   - COMPUTES ZONAL COVARIANCE OF TWO GRID SETS                 2 1
   zondev   - COMPUTES ZONAL DEVIATIONS OF A GRID SET                    2 1

   zonvavg  - COMPUTES MULTI-LEVEL ZONAL AVERAGE OF A SET OF WINDS       2 2
